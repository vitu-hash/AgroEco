# 🌱 AgroEco

## Sobre o Projeto

O **AgroEco** é um site educativo desenvolvido com o objetivo de promover a conscientização sobre a importância da sustentabilidade na agricultura. A plataforma apresenta conteúdos informativos sobre práticas agrícolas sustentáveis e oferece um quiz interativo para estimular o aprendizado de forma dinâmica e envolvente.

O projeto busca demonstrar como a tecnologia pode ser utilizada para disseminar conhecimentos sobre preservação ambiental, produção responsável e desenvolvimento sustentável no campo.

---

## 🎯 Objetivos

- Incentivar a adoção de práticas agrícolas sustentáveis.
- Informar sobre a relação entre agricultura e meio ambiente.
- Promover a educação ambiental de forma acessível.
- Estimular o aprendizado por meio de um quiz interativo.

---

## ✨ Funcionalidades

- Página inicial com apresentação do tema.
- Conteúdo educativo sobre sustentabilidade e agricultura.
- Seção com práticas sustentáveis aplicadas ao campo.
- Explicação dos benefícios da agricultura sustentável.
- Quiz interativo com perguntas de múltipla escolha.
- Sistema de pontuação automática.
- Exibição de resultado personalizado conforme o desempenho.
- Botão para reiniciar o quiz.
- Layout responsivo para computadores, tablets e smartphones.
